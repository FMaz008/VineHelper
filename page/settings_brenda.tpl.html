Vine Helper seamlessly integrates with the (unofficial) Amazon Vine Discord server's Brenda bot. Upon joining the server and linking your account, you gain the ability to share products with the bot to recommend to users in your country.
This feature exchanges data (API token, country, queue, asin, etv, etc.) with the Brenda API server for Discord.
Step 1. Join the Discord Server Join the Discord server: https://discord.gg/amazonvine
Step 2. Obtain Your API-Token
  • Go to any channel on the server and type the command:
    /api-token
  • A bot named Karen will reply with an API token.
  • Copy & Paste the API Token into Step 3.
