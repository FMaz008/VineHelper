Version {{$appVersion}} change notes:
The notification monitor was overhauled internally, a long overdue task required from the growing set of features complicating the code. VH now tries to tracks the vine tier availability of items and implemented a concept of devices to complement the UUID system.


Change log:
  • NEW: Added link next to the ASIN in the saved reviews list.
  • NEW: Device name and id VH now allow you to set a custom device name, so you can identify which device are linked to a specific UUID. Additionally, each device now gets a unique(-ish) id (an anonymous fingerprint containing no data) that is used to identify the device alongside the UUID.
  • NEW: Details icon in the toolbar. Clicking on it will open a popup with the item's details.
  • CHANGE: Unavailable tab now regroup the items which have received an ITEM_NOT_IN_ENROLLMENT error, and no longer rely on the successful/unsuccessful order counts. This behavior is now in line with the notifications monitor.
  • FIX: mobile view. The highlight toolbar was on the same line as the RFY/AFA/AI buttons. It is now on a new line.
  • FIX: Modal/popup closing logic. Modal/popup closing logic has fixed so pressing ESC closes the top one.
  • FIX: Rogue error. Rogue error on page without items (ie: empty AFA)
  • FIX: The Available tab had a column gap between items, causing the grid to look different than the other tabs.
  • FIX: Unescaping titles has been improved for vine searches and tooltips.
  • FIX: Display error message when trying to save a review without a title.
  • REMOVED: Reliability score. User's reliability score is no longer displayed.
  • Notification Monitor:
    • NEW: Auto-refresh. When using the notification monitor, VH will automatically load a vine listing page every 5 to 10 minutes. You may see a tab opening and closing promptly in the background when that occurs.
    • NEW: If you are using Open in Place,CTRL+Click on a See Details button will open it in a new tab.
    • NEW: Highlight item with Unknown ETV. New option to highlight items with an unknown ETV.
    • NEW: Bump 0ETV. New option to bump 0ETV items to the top of the list. (It was a default behavior before)
    • NEW: Clear unavailable Notification Monitors now have a button to clear unavailable items.
    • NEW: Sort by ETV. Notification Monitors now have a sort by ETV feature.
    • NEW: Silver items detection. Items exceeding the silver limit but still showing to silver members are now flagged as silver items and available to silver members.
    • NEW: Hide gold notifications. Silver tier users can now hide gold notifications from the notification monitor.
    • CHANGE: Clear all now clear visible items only.
    • CHANGE: Notification monitor header design. Slight change to the header design to make it more scalable with various screen sizes.
    • IMPROVEMENT: Performance improvement. Auto truncation, and other stuff.
    • FIX: Clarified error message when the notification monitor is not activated in VH's settings.
    • FIX: Unresponsive clicks after certain items were removed in bulk.
  • Premium feature(s):
    • NEW: Variants button. Premium tier 2 and above; On the listing pages, adds a button to view the known variants of an item and load the variant directly in the See Details' window. Bypass the need for a spinner fix or to do the manual editing of the See Details button.
