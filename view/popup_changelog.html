Change log:

General:

  • FIX: Exporting local hidden items would export pinned items
  • FIX: Firefox context menu would not add keywords to their respective list.
  • FIX: Multiple connections to the websocket server could be opened at the same time, causing duplicate on screen (toaster) notifications.
  • FIX: Backward support for Firefox pre-121.
  • FIX: Search bar too wide when using mobile stylesheets.
  • FIX: Ignore single letter words when clicking the search icons.

Notification Monitor:

  • NEW: Placeholders. Placeholders are used to prevent new items from shifting the entire grid of items everytime something is added. With placeholders, items keep their horizontal placement in the grid, making it easier to browse.

    Note: Placeholders are not "hidding" items.
  • NEW: Option to ignore unknown/0ETV coloring if the item match with a highlight keyword.
  • CHANGE: Auto-Load default mode is silent on new installs.
